# linux网络管理

## 读取网卡信息文件

```shell
vi /etc/sysconfig/network-scripts/ifcfg-eth0
```

## 读取主机名文件

```shell
vi /etc/sysconfig/network
```



## 修改DNS文件

```shell
vi /etc/resolv.conf
```





```shell
ifconfig 查看配置文件
```

配置ip 子网掩码 网关 dns才能上网

## 常用命令

```shell
ifconfig #查看与临时配置网络状态
ifdown device name #关闭网卡
ifup device name #开启网卡
netstat #查询网络状态
nslookup #域名解析 进行域名与ip地址解析
nslookup server #查看dns
```
